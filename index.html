Step 1

The image to above is a solved puzzle. We can use numbers to reference each tile.
A solved puzzle will have all of the tiles ordered from left to right in each row with the empty space in the bottom right corner.
You should begin by placing tiles 1, 2, 3, 4, and 5 in their correct position.


Step 2

One possible way to achieve this is to position the tiles like the image above.
Try not to move tiles 1, 2, or 3. Tile 5 should be directly underneath tile 4.


Step 3

Now, the board should look something like this after sliding tiles 4 and 5 in the correct position.
The same method can be used to solve rows 2 and 3.


Step 4

The board should look something like this after solving rows 1, 2, and 3.


Step 5

The same method used for rows 1, 2 and 3 must now be done per column for row 4 and 5.
Do this for the first 3 columns from left to right.
The last 3 tiles should now easily move in place and you are finished!
